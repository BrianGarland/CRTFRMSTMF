**What is CRTFRMSTMF?** 

CRTFRMSTMF is a wrapper over the IBM i CRTxxx compile commands that do not allow a stream file.


**Download with RELIC:**

```
RELICGET PLOC('https://github.com/BrianGarland/CRTFRMSTMF/archive/master.zip') 
PDIR('CRTFRMSTMF') PNAME(#CRTFRMSTM)
```
